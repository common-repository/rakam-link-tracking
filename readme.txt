== Description ==
An Extension for WordPress that allows you to Link Tracking.

1. Just install the plugin and the Digit Link Tracking admin section will be created in WordPress' settings page.
2. Enter the names of the URL parameters you want to be tracked
3. That's it. The URL parameters you add will be included in all links.

== Installation ==

1. Upload `rakam-link-tracking` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done go to Rakam Link Tracking section in the settings menu and start using
